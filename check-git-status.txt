# VNMaterial - Quick Check Commands

## 1. Kiểm tra Git đã cài đặt chưa
git --version

## 2. Kiểm tra trạng thái Git
cd /d "c:\xampp\htdocs\vnmt"
git status

## 3. Kiểm tra remote repository
git remote -v

## 4. Kiểm tra log commit gần nhất
git log --oneline -5

## 5. Kiểm tra có file nào chưa commit
git status --porcelain

## 6. Test push (nếu cần)
git push origin main
